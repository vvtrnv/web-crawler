# web-crawler

## Versions
`node v18.15.0^` `npm v9.5.0^`

## Steps to run this project:
1. Run `npm i` command
2. Setup database settings inside `data-source.ts` file
3. Run `npm start` command


### Environment variables
Path `./config/.env`
example:
```json
BASE_URL=https://history.eco/
DATABASE=web-crawler
```
